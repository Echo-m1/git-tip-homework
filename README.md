# GitTips (–ß–∞—Å—Ç—å 1. –û—Ç –æ—Ç–∫—Ä—ã—Ç–∏—è –∫–æ–Ω—Å–æ–ª–∏ –¥–æ –ø–µ—Ä–≤–æ–≥–æ –∫–æ–º–º–∏—Ç–∞)

## –í–≤–æ–¥–Ω–∞—è

---

–ö–æ–º–∞–Ω–¥–Ω–∞—è —Å—Ç—Ä–æ–∫–∞ ( `CLI` ‚Äì **C**omand-**l**ine **I**nterface ) ‚Äì —Ç–µ–∫—Å—Ç–æ–≤—ã–π –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å, —Ç–µ—Ä–º–∏–Ω–∞–ª, –∫–æ–Ω—Å–æ–ª—å. 

–¢–µ–∫—Å—Ç–æ–≤—ã–π –∞–Ω–∞–ª–æ–≥, —Ç–æ—á–Ω–µ–µ —Å–∫–∞–∑–∞—Ç—å –ø—Ä–∞—Ä–æ–¥–∏—Ç–µ–ª—å –ø—Ä–∏–≤—ã—á–Ω–æ–≥–æ –≤—Å–µ–º –Ω–∞–º –≥—Ä–∞—Ñ–∏—á–µ—Å–∫–æ–≥–æ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–∞. –ö–æ–≥–¥–∞-—Ç–æ –æ—á–µ–Ω—å –¥–∞–≤–Ω–æ –µ–≥–æ –Ω–µ –±—ã–ª–æ –∏ —É–ø—Ä–∞–≤–ª–µ–Ω–∏–µ –ø—Ä–æ–≥—Ä–∞–º–º–∞–º–∏ –∫–æ–º–ø—å—é—Ç–µ—Ä–∞ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏–ª–æ —Å –ø–æ–º–æ—â—å—é –∫–æ–º–∞–Ω–¥–Ω–æ–π —Å—Ç—Ä–æ–∫–∏. –°–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ –∫–æ–º–ø—å—é—Ç–µ—Ä–æ–º –º–æ–≥–ª–∏ –ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è —Ç–æ–ª—å–∫–æ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Å—Ç—ã.

–°–µ–π—á–∞—Å –∫–æ–º–∞–Ω–¥–Ω–∞—è —Å—Ç—Ä–æ–∫–∞ —Ä–µ–¥–∫–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –ª—é–¥—å–º–∏, –∫–æ—Ç–æ—Ä—ã–µ "–Ω–µ –≤–æ—à–ª–∏ –≤ –∞–π—Ç–∏", –Ω–æ –¥–ª—è –ø—Ä–æ–≥—Ä–∞–º–º–∏—Å—Ç–æ–≤ ‚Äì **–Ω–µ–æ–±—Ö–æ–¥–∏–º—ã–π** –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç.

---

–ì—Ä–∞—Ñ–∏—á–µ—Å–∫–∏–π –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å (`GUI` ‚Äì **G**raphical **U**ser **I**nterface) ‚Äì –æ–∫–Ω–∞, –∑–Ω–∞—á–∫–∏, –∫–Ω–æ–ø–∫–∏, –≤—ã–ø–∞–¥–∞—é—â–∏–µ –º–µ–Ω—é —Å –Ω–∞—Å—Ç—Ä–æ–π–∫–∞–º–∏ –∏ –ø—Ä–æ—á–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã. –ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è –∫–æ–º–ø—å—é—Ç–µ—Ä–æ–º –∏ –ø—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏ –≤—Å–µ–º–∏ –µ–≥–æ –ø—Ä–æ–≥—Ä–∞–º–º–∞–º–∏. –¢–µ—Ö–Ω–∏—á–µ—Å–∫–∏ —Ä–∞–±–æ—Ç–∞–µ—Ç —Ç–∞–∫ –∂–µ –∫–∞–∫ –∏ –∫–æ–º–∞–Ω–¥–Ω–∞—è —Å—Ç—Ä–æ–∫–∞, –Ω–æ –¥–µ–ª–∞–µ—Ç —ç—Ç–æ –ø—Ä–æ—â–µ –∏ –¥–æ—Å—Ç—É–ø–Ω–µ–µ.  

---

VCS ( Version Control System ‚Äì '—Å–∏—Å—Ç–µ–º–∞ –∫–æ–Ω—Ç—Ä–æ–ª—è –≤–µ—Ä—Å–∏–π' ), SCM ( Source Control Management ‚Äì '—Å–∏—Å—Ç–µ–º–∞ —É–ø—Ä–∞–≤–≤–ª–µ–Ω–∏—è –∏—Å—Ö–æ–¥–Ω—ã–º –∫–æ–¥–æ–º' ) ‚Äì –ø—Ä–æ–≥—Ä–∞–º–º–Ω–æ–µ –æ–±–µ—Å–ø–µ—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –ø–æ–º–æ–≥–∞–µ—Ç –æ—Ç—Å–ª–µ–∂–∏–≤–∞—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ –ø—Ä–æ–≥—Ä–∞–º–º–∞—Ö, —Ç–µ–∫—Å—Ç–æ–≤—ã—Ö —Ñ–∞–π–ª–∞—Ö, –±–æ–ª—å—à–∏—Ö —Ñ–∞–π–ª–∞—Ö, –≤–µ–±-—Å–∞–π—Ç–∞—Ö –∏ —Ç.–¥.

–û–¥–Ω–æ –∏–∑–º–µ–Ω–µ–Ω–∏–µ –∏–ª–∏ –≥—Ä—É–ø–ø–∞ –∏–∑–º–µ–Ω–µ–Ω–∏–π –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è —Ä–µ–≤–∏–∑–∏–µ–π/–≤–µ—Ä—Å–∏–µ–π.
–ö–∞–∂–¥–∞—è —Ç–∞–∫–∞—è –≤–µ—Ä—Å–∏—è —Å–æ–¥–µ—Ä–∂–∏—Ç –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ —Ç–æ–º, —á—Ç–æ –∏–∑–º–µ–Ω–∏–ª–æ—Å—å, –∫—Ç–æ –≤–Ω–µ—Å –∏–∑–º–µ–Ω–µ–Ω–∏—è, –∫–æ–≥–¥–∞ —ç—Ç–æ –±—ã–ª–æ –∏, –∏–Ω–æ–≥–¥–∞, –∫–æ–º–º–µ–Ω—Ç–∞—Ä–∏–∏ –∫ –∏–∑–º–µ–Ω–µ–Ω–∏—é.

---

 ### –ö–æ–º–∞–Ω–¥–Ω–∞—è —Å—Ç—Ä–æ–∫–∞

–ò—Ç–∞–∫, –¥–ª—è –¥–∞–ª—å–Ω–µ–π—à–µ–π —Ä–∞–±–æ—Ç—ã —Å `Git` –Ω–∞–º –ø–æ–Ω–∞–¥–æ–±–∏—Ç—Å—è –æ–∑–Ω–∞–∫–æ–º–∏—Ç—å—Å—è –∫–∞–∫ –º–∏–Ω–∏–º—É–º —Å –Ω–∏–∂–µ–ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω–Ω—ã–º–∏ –∫–æ–º–∞–Ω–¥–∞–º–∏. –û–Ω–∏ –ø–æ–∑–≤–æ–ª—è—Ç –Ω–∞–º –ø–µ—Ä–µ–º–µ—â–∞—Ç—å—Å—è –ø–æ —Ñ–∞–π–ª–æ–≤–æ–π —Å–∏—Å—Ç–µ–º–µ, –ø—Ä–æ—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å —Å–æ–∑–¥–∞–≤–∞—Ç—å –∏ —É–¥–∞–ª—è—Ç—å —Ñ–∞–π–ª—ã.

---

_–ù—É–º–µ—Ä–æ–≤–∞–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫ ‚Äì –∫–æ–º–∞–Ω–¥—ã._

_–ù–µ–Ω—É–º–µ—Ä–æ–≤–∞–Ω–Ω—ã–π ‚Äì –∏—Ö –∫–ª—é—á–∏_

_–ö–ª—é—á ‚Äì —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–π –∞—Ä–≥—É–º–µ–Ω—Ç, –∫–æ—Ç–æ—Ä—ã–π –ø–∏—à–µ—Ç—Å—è —á–µ—Ä–µ–∑ –ø—Ä–æ–±–µ–ª –∏ –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å –æ–¥–Ω–æ–≥–æ –∏–ª–∏ –¥–≤—É—Ö –¥–µ—Ñ–∏—Å–æ–≤. –û–Ω –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –ø–∞—Ä–∞–º–µ—Ç—Ä—ã, –∫–æ—Ç–æ—Ä—ã–µ –±—É–¥—É—Ç –ø—Ä–∏–º–µ–Ω–µ–Ω—ã –∫ –∫–æ–º–∞–Ω–¥–µ_

---

1. **`pwd`** ‚Äì ( **print working directory** ‚Äì '–ø–æ–∫–∞–∑–∞—Ç—å —Ä–∞–±–æ—á—É—é –ø–∞–ø–∫—É' ) ‚Äì –≤—ã–≤–æ–¥–∏—Ç –ø—É—Ç—å –∫ —Ç–µ–∫—É—â–µ–π –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏. –ï—Å–ª–∏ –≤—ã –Ω–µ –≤–∏–¥–∏—Ç–µ –ø–æ–¥—Å–∫–∞–∑–æ–∫ –∫–æ–Ω—Å–æ–ª–∏, –∏–ª–∏ –ø—Ä–æ—Å—Ç–æ –∑–∞–±—ã–ª–∏ –ø—Ä–æ –Ω–∏—Ö, –∏–ª–∏ —Ç–∞–∫ –≥–ª—É–±–æ–∫–æ –∑–∞–ª–µ–∑–ª–∏ –≤ —Ñ–∞–π–ª–æ–≤—É—é —Å–∏—Å—Ç–µ–º—É, —á—Ç–æ –ø—É—Ç—å –∫ –≤–∞—à–µ–º—É —Ç–µ–∫—É—â–µ–º—É –ø–æ–ª–æ–∂–µ–Ω–∏—é –Ω–µ –ø–æ–º–µ—â–∞–µ—Ç—Å—è –≤ –ø–æ–ª–µ –¥–ª—è –ø–æ–¥—Å–∫–∞–∑–∫–∏, —Ç–æ **—ç—Ç–∞ –∫–æ–º–∞–Ω–¥–∞** ‚Äì —Ç–æ, —á—Ç–æ –≤–∞–º –Ω—É–∂–Ω–æ

---

2. **`cd`** ‚Äì ( **change directory** ‚Äì '—Å–º–µ–Ω–∏—Ç—å –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏—é' ) ‚Äì –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –Ω–∞ –≤—Ö–æ–¥ –æ–¥–∏–Ω –ø–∞—Ä–∞–º–µ—Ç—Ä: –Ω–∞–∑–≤–∞–Ω–∏–µ –ø–∞–ø–∫–∏, –≤ –∫–æ—Ç–æ—Ä—É—é –≤—ã —Ö–æ—Ç–∏—Ç–µ "—Å–ø—É—Å—Ç–∏—Ç—å—Å—è", –∏–ª–∏ —Ü–µ–ª—ã–π –ø—É—Ç—å. –¢—É—Ç –æ–ø–∏—Å–∞–Ω–∏–µ –Ω–µ —Ç–∞–∫–æ–µ –¥–ª–∏–Ω–Ω–æ–µ, –Ω–æ —Å–µ–π—á–∞—Å –º—ã –ø–æ–∑–Ω–∞–∫–æ–º–∏–º—Å—è —Å –∫–ª—é—á–∞–º–∏:
- **`.`** ‚Äì –æ–±—Ä–∞—Ç–∏—Ç—å—Å—è –∫ —Ç–µ–∫—É—â–µ–π –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏ ( –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–æ–≤–æ–ª—å–Ω–æ —Ä–µ–¥–∫–æ, –Ω–∞–ø—Ä–∏–º–µ—Ä –µ—Å–ª–∏ —Å–∫—Ä–∏–ø—Ç –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –Ω–∞ –≤—Ö–æ–¥ –ø–∞–ø–∫—É ). `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: cd .` –ï—Å–ª–∏ –≤—ã –Ω–∏–∫–∞–∫–æ–≥–æ —Ç–∞–∫–æ–≥–æ —Å–∫—Ä–∏–ø—Ç–∞ –Ω–µ –∑–∞–ø—É—Å–∫–∞–ª–∏, —Ç–æ —ç—Ñ—Ñ–µ–∫—Ç–∞ –æ—Ç –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è —ç—Ç–æ–π –∫–æ–º–∞–Ω–¥—ã –Ω–∏–∫–∞–∫–æ–≥–æ –Ω–µ –±—É–¥–µ—Ç, –º–æ–∂–µ—Ç–µ –ø–æ–ø—Ä–æ–±–æ–≤–∞—Ç—å
- **`~`** ‚Äì –ø–µ—Ä–µ—Ö–æ–¥ –≤ –¥–æ–º–∞—à–Ω—é—é –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏—é ( –ù–µ –ø—É—Ç–∞—Ç—å —Å –∫–æ—Ä–Ω–µ–≤—ã–º –∫–∞—Ç–∞–ª–æ–≥–æ–º! –î–æ–º–∞—à–Ω—è—è –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏—è –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏—Ö —Ñ–∞–π–ª–æ–≤: —É –≤–∞—Å —Ç–∞–º –Ω–∞—Ö–æ–¥—è—Ç—Å—è –≤—Å–µ —Ñ–æ—Ç–æ–≥—Ä–∞—Ñ–∏–∏, –≤–∏–¥–µ–æ, –ø–∞–ø–∫–∏ —Å –∏–≥—Ä–∞–º–∏, –º–µ–º–∞–º–∏ –∏ –Ω–µ–±–µ–∑—ã–∑–≤–µ—Å—Ç–Ω—ã–π —Ä–∞–±–æ—á–∏–π —Å—Ç–æ–ª ). `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: cd ~` 
- **`Desktop`** ‚Äì –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–µ—Ä–µ–π—Ç–∏ –≤ –ø–∞–ø–∫—É Desktop ( –µ—Å–ª–∏ —Ç–∞–∫–∞—è –µ—Å—Ç—å –≤ —Ç–µ–∫—É—â–µ–π –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏ ). `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: cd Desktop` 
- **`Desktop/games`** ‚Äì –∞ —Ç–∞–∫ –º—ã –ø–µ—Ä–µ–π–¥–µ–º –≤ –ø–∞–ø–∫—É games, –∫–æ—Ç–æ—Ä–∞—è –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –≤ –ø–∞–ø–∫–µ Desktop. `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: cd Desktop/games`
- **`..`** ‚Äì –ø–µ—Ä–µ—Ö–æ–¥ –Ω–∞ –æ–¥–∏–Ω —É—Ä–æ–≤–µ–Ω—å –≤–≤–µ—Ä—Ö. `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: cd ..`
- **`/`** ‚Äì –ø–µ—Ä–µ—Ö–æ–¥ –≤ –∫–æ—Ä–Ω–µ–≤—É—é –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏—é. –ö–æ—Ä–Ω–µ–≤–∞—è –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏—è ‚Äì –≥–ª–∞–≤–Ω–∞—è –ø–∞–ø–∫–∞ –Ω–∞ –∫–æ–º–ø—å—é—Ç–µ—Ä–µ –∏–ª–∏ —Å–µ—Ä–≤–µ—Ä–µ. –í –Ω–µ–π —Ö—Ä–∞–Ω—è—Ç—Å—è –≤–æ–æ–±—â–µ –≤—Å–µ —Ñ–∞–π–ª—ã, –∫–∞–∫ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–µ, —Ç–∞–∫ –∏ —Å–∏—Å—Ç–µ–º–Ω—ã–µ. –ü–æ–ª—å–∑—É–π—Ç–µ—Å—å —Å –æ—Å—Ç–æ—Ä–æ–∂–Ω–æ—Å—Ç—å—é, —á—Ç–æ–±—ã –Ω–µ –∏–∑–º–µ–Ω–∏—Ç—å —á—Ç–æ-—Ç–æ –Ω–µ –¥–∞–π –±–æ–≥. –Ø, –Ω–∞–ø—Ä–∏–º–µ—Ä, –Ω–µ –∑–Ω–∞—é –∫–∞–∫ –≤ —Å–ª—É—á–∞–µ —á–µ–≥–æ –≤—Å–µ –æ–±—Ä–∞—Ç–Ω–æ –æ—Ç–∫–∞—Ç–∏—Ç—å.  `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: cd /`

---

3. **`ls`** ‚Äì ( **list directory contents** ‚Äì '–æ—Ç–æ–±—Ä–∞–∑–∏—Ç—å —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–π' ) ‚Äì –¥–∞–Ω–Ω–∞—è –∫–æ–º–∞–Ω–¥–∞ –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç –∫–∞–∫–∏–µ –≤–ª–æ–∂–µ–Ω–Ω—ã–µ –ø–∞–ø–∫–∏ —Å–æ–¥–µ—Ä–∂–∏—Ç —Ç–µ–∫—É—â–∞—è –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏—è
- **`-a`** ‚Äì –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç —Ä–∞—Å—à–∏—Ä–µ–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫ ( –≤–∫–ª—é—á–∞—è —Å–∫—Ä—ã—Ç—ã–µ —Ñ–∞–π–ª—ã ). `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: ls -a`
- **`~`** ‚Äì –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ –¥–æ–º–∞—à–Ω–µ–π –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏. `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: ls ~`
- **`..`** ‚Äì –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ —Ä–æ–¥–∏—Ç–µ–ª—å—Å–∫–æ–π –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏. `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: ls ..`
- **`l`** ‚Äì –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ –≤ –≤–∏–¥–µ —Å–ø–∏—Å–∫–∞. `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: ls -l`

---

4. **`touch`** ‚Äì ( –∫–æ—Å–Ω—É—Ç—å—Å—è ) ‚Äì —Å–æ–∑–¥–∞—Ç—å **—Ñ–∞–π–ª**. `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: touch test.txt`
- **`~`** ‚Äì —Å–æ–∑–¥–∞—Ç—å —Ñ–∞–π–ª –≤ –¥–æ–º–∞—à–Ω–µ–π –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏. `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: touch ~ test.txt (–∏–ª–∏ touch ~/test.txt)`
- **`..`** ‚Äì —Å–æ–∑–¥–∞—Ç—å —Ñ–∞–π–ª –≤ –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏ –Ω–∞ –æ–¥–∏–Ω —É—Ä–æ–≤–µ–Ω—å –≤—ã—à–µ —Ç–µ–∫—É—â–µ–π. `–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: touch ../test.txt`

---
